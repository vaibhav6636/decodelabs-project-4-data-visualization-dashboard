# Data Visualization Dashboard

## Project Overview

This project focuses on creating an interactive Power BI dashboard to analyze sales performance, customer behavior, product demand, and business trends using an e-commerce dataset.

The dashboard transforms raw transactional data into meaningful insights through KPI cards, charts, and interactive filters.

## Dashboard Preview

![Dashboard Preview](Images/E-Commerce%20Sales%20Performance%20Dashboard.png)

---

## Objective

The objective of this project is to create visual representations of data that help identify patterns, trends, and business insights for better decision-making.

---

## Tools Used

- Power BI
- Microsoft Excel

---

## Dataset Information

- Total Records: 1,200
- Time Period: January 2023 – June 2025
- Dataset Type: E-Commerce Sales Dataset

---

## Dashboard Components

### KPI Cards
- Total Orders
- Total Revenue
- Average Order Value
- Total Products Sold

### Visualizations
- Revenue by Product
- Orders by Product
- Orders by Payment Method
- Orders by Referral Source
- Monthly Order Trend
- Order Status Distribution

### Interactive Filters
- Product
- Payment Method
- Order Status
- Referral Source

---

## Key Insights

- Chair generated the highest revenue.
- Printer recorded the highest number of orders.
- Online payment was the most preferred payment method.
- Instagram generated the highest customer referrals.
- June recorded the highest order volume.
- Cancelled and returned orders were relatively high.

---

## Recommendations

- Focus marketing efforts on high-performing products.
- Reduce order cancellations and returns through better order management.
- Continue investing in effective marketing channels.
- Use sales trends for inventory and business planning.
- Monitor business performance regularly using dashboards.

---

## Project Outcome

The dashboard provides a clear view of sales performance, customer behavior, and business trends, helping support data-driven decision-making.
